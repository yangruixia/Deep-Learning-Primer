# 1. Flask简介
Flask是一个由Python语言写成的轻量级Web框架。Flask最显著的特点是它是一个“微”框架，轻便灵活，但同时又易于扩展。

Flask非常适用于科研成果展示平台的搭建。

# 2. Flask开发入门
Flask基于Python和基础HTML。在学习Flask之前必须了解Python基本语法和基础网页开发。
